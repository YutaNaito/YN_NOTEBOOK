Linuxコマンド¶

source¶

ファイルに書かれたコマンドを現在のシェルで実行するというコマンド。シェルの設定ファイルを反映させる際に使用。このコマンドは現在のシェルで実行される。bash ファイルは別のシェルで実行される

  • source <ファイル> [ファイル実行時の引数]

  • . <ファイル> [ファイル実行時の引数]

【source】コマンド / 【.】コマンド---シェルの設定を即座に反映させる
